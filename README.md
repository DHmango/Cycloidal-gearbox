### What actually is this?

This is a compact 14:1 reducer using cycloidal drive. It is intended to be used on motors with 1/8" D output shafts, such as those used in First Tech Challenge.

### Why did I make this?

I have sometimes seen cool gear reducers online. Cycloidal drives in particular have always fascinated me because they move mesmerisingly and smoothly despite usinig eccentric motion. So, I wanted to see if I could make one myself. After making some plastic demo versions, I wanted to take it to the next level with a real metal version. The current version uses 46 tiny bearings and is made of aluminum. Also, it should be better in many ways than comparable normal gear reducers.

### How to use this?

You have to buy all the aliexpress things listed in the bill of materials, print out the plastic things, and also get the metal parts cut. I suggest trying several sheet metal cutting services for the lowest price. Once you've bought all the parts, you have to go through a tedious assembly proccess which is detailed... no where yet. When I assemble mine, I will make instructions to go with it.
